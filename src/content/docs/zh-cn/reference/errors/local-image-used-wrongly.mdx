> **LocalImageUsedWrongly**: `Image`'s 和 `getImage` 的 `src` 参数必须是一个导入的图片或一个 URL，不能是一个文件路径。接收到了 `IMAGE_FILE_PATH`。

## 哪里发生了错误？

当使用默认的图片服务时，`Image` 和 `getImage` 的 `src` 参数必须是一个导入的图片或一个 URL，不能是一个文件路径。

```astro
---
import { Image } from "astro:assets";
import myImage from "../my_image.png";
---

<!-- 对的: `src` 是完整的导入的图片 -->
<Image src={myImage} alt="Cool image" />

<!-- 错的: `src` 是一个图片的 `src` 路径而不是完整的图片 -->
<Image src={myImage.src} alt="Cool image" />
```
