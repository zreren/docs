在这个单元中，你将使用 **Astro 群岛**将前端框架组件引入你的 Astro 网站。

你将会：
- 为你的 Astro 项目添加一个 UI 框架：Preact
- 使用 Preact 创建一个交互式的问候组件
- 了解在何时可能不选择岛屿进行交互
